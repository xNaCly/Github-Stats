# minimal github api wrapper

### Docs:

1. Class init:

```js
const github = new Github("<orgaName>","<OAuthToken>")

--> void
```

-   arguments:
    -   OrgaName or Username: `fosscord`/`xnacly`
-   OAuth token should have `repo` & `user` scopes
